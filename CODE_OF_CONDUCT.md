# Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

We are committed to providing a welcoming and inclusive environment for everyone. Please be respectful and constructive in all interactions.

## Reporting

If you experience or witness unacceptable behavior, please open an issue on the [quick-question repo](https://github.com/tykisgod/quick-question/issues) or contact the maintainer directly.
