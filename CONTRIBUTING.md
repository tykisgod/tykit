# Contributing to tykit

> **This repository is an automated mirror.** Do not open PRs or push changes here. All development happens in [tykisgod/quick-question](https://github.com/tykisgod/quick-question) under `packages/com.tyk.tykit/`.

## How to Contribute

1. Fork and clone [quick-question](https://github.com/tykisgod/quick-question)
2. Make changes in `packages/com.tyk.tykit/`
3. Run `./test.sh` from the qq repo root
4. Open a PR against `tykisgod/quick-question`

Changes merged to qq's `main` branch are automatically synced to this repo via CI.

## Reporting Issues

Please open issues on the [quick-question repo](https://github.com/tykisgod/quick-question/issues) with the `tykit` label.
